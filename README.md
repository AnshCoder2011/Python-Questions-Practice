# 🐍 Python Practice – Week 1 (Day 1)

This repository contains my **Day 1 Python practice problems** as part of my Python learning journey.  
The goal of this practice set is to **strengthen programming fundamentals and problem-solving skills**.

All programs are written in **simple beginner-friendly Python** without relying heavily on built-in functions so the core logic is easy to understand.

---

## 📚 Topics Covered

During this practice session I worked on problems related to:

- Variables and input/output
- Conditional statements (`if-else`)
- Loops (`for`, `while`)
- Basic mathematics in programming
- String manipulation
- Number-based problems
- Simple algorithmic thinking

## 📝 Questions Practiced

### 1️⃣ Swap Two Variables
A program that swaps the values of two variables.

### 2️⃣ User Input Program
Takes a name as input and displays a greeting message.

### 3️⃣ Even or Odd Checker
Checks whether a given number is even or odd.

### 4️⃣ Multiplication Table
Prints the multiplication table of a given number.

### 5️⃣ Greatest Among Three Numbers
Finds the largest number among three user inputs.

### 6️⃣ Skipped
This question was skipped because it was formula-based and already covered by earlier concepts.

### 7️⃣ Factorial Program
Calculates the factorial of a number using a loop.

### 8️⃣ Vowel Counter
Counts the number of vowels present in a given string.

### 9️⃣ Reverse a String
Reverses a string without using slicing.

### 🔟 Palindrome Checker
Checks whether a string is a palindrome or not.

### 1️⃣1️⃣ Sum of First N Natural Numbers
Calculates the sum of numbers from `1` to `N`.

### 🎮 Number Guessing Game
A simple guessing game where the user guesses a randomly generated number.

### 1️⃣3️⃣ Prime Numbers Between 1 and 100
Prints all prime numbers between 1 and 100.

### 1️⃣4️⃣ Leap Year Checker
Checks whether a given year is a leap year.

### 1️⃣6️⃣ GCD Finder
Finds the Greatest Common Divisor of two numbers.

### 1️⃣7️⃣ LCM Finder
Finds the Least Common Multiple of two numbers.

### 1️⃣9️⃣ Sum of Digits
Calculates the sum of digits of a number.

### 2️⃣0️⃣ Second Largest Number in a List
Finds the second largest number in a list **without using built-in functions**.
